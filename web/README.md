# TradeLens Web Frontend

React + TypeScript + Vite frontend for the TradeLens local trading platform.

## Features

- Live candlestick updates from WebSocket channels.
- Footprint chart rendered with PixiJS (WebGL).
- Orderbook + trading panel for paper trading.
- Backtest execution dashboard with metrics and trade table.
- Zustand stores for market, order, and UI state.

## Run

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Test

```bash
npm run test
```
